# بات تلگرام + پخش‌کننده فیلم (بدون VPS)

## چطور کار می‌کنه؟
1. فیلم رو به یه بات کوچیک تلگرام می‌فرستی.
2. بات، فیلم رو داخل تلگرام به یه چت رله (بین بات و اکانت شخصی/یوزربات) فوروارد می‌کنه — این کار سمت سرور تلگرامه و محدودیت حجم نداره.
3. بات یه لینک بهت می‌ده: `https://appشما.vercel.app/watch.html?id=...`
4. وقتی این لینک رو تو مرورگر تلویزیون باز می‌کنی، یه فانکشن روی Vercel با اکانت شخصی (نه بات) به تلگرام وصل میشه و فیلم رو تیکه‌تیکه براش استریم می‌کنه — چون از طریق اکانت واقعیه، محدودیت ۲۰ مگابایتی باتا رو نداره.

**پیشنهاد:** برای مرحله ۲ و ۳ یه اکانت تلگرام تازه (نه اکانت اصلی خودت) بساز و باهاش یوزربات رو راه بنداز، امن‌تره.

---

## مرحله ۱: گرفتن API ID و API Hash
1. برو به [my.telegram.org](https://my.telegram.org) و با شماره‌ای که می‌خوای یوزربات باشه وارد شو.
2. برو تو بخش **API Development Tools**.
3. یه اپ بساز (اسم و توضیحات فرقی نداره) و `api_id` و `api_hash` رو یادداشت کن.

## مرحله ۲: ساخت بات رله در BotFather
1. تو تلگرام برو سراغ **@BotFather**.
2. دستور `/newbot` رو بزن، اسم و یوزرنیم بده.
3. توکنی که می‌ده رو یادداشت کن (`TELEGRAM_BOT_TOKEN`).
4. با همون اکانتی که قراره یوزربات باشه (همون شماره مرحله ۱)، به بات جدید `/start` بزن.

## مرحله ۳: نصب و لاگین محلی (فقط یه بار، روی کامپیوتر خودت)
```bash
npm install
TG_API_ID=xxxx TG_API_HASH=xxxx node scripts/login.js
```
شماره تلفن، کد تایید و (اگه داری) رمز دو مرحله‌ای رو وارد کن. در آخر یه رشته طولانی به اسم **Session String** می‌گیری — این رو جایی امن نگه دار، معادل رمز اکانتته.

## مرحله ۴: پیدا کردن RELAY_CHAT_ID
بعد از `/start` زدن به بات، یه پیام (مثلا "سلام") از همون اکانت به بات بفرست، بعد این آدرس رو تو مرورگر باز کن (به‌جای TOKEN توکن باتت رو بذار):
```
https://api.telegram.org/botTOKEN/getUpdates
```
تو خروجی JSON دنبال `"chat":{"id": ...}` بگرد — همون عدد `RELAY_CHAT_ID`ـه.

## مرحله ۵: دیپلوی روی Vercel
1. این پروژه رو به یه ریپازیتوری گیت‌هاب پوش کن (یا مستقیم با `vercel` CLI دیپلوی کن).
2. تو داشبورد Vercel، پروژه رو وارد کن و متغیرهای محیطی زیر رو ست کن (از `.env.example` الگو بگیر):
   - `TG_API_ID`
   - `TG_API_HASH`
   - `SESSION_STRING`
   - `TELEGRAM_BOT_TOKEN`
   - `RELAY_CHAT_ID`
   - `PUBLIC_BASE_URL` → بعد از اولین دیپلوی، آدرسی که Vercel می‌ده رو اینجا بذار و دوباره دیپلوی کن.

## مرحله ۶: وصل کردن Webhook بات به Vercel
یه بار این آدرس رو تو مرورگر باز کن (TOKEN و DOMAIN رو جایگزین کن):
```
https://api.telegram.org/botTOKEN/setWebhook?url=https://appشما.vercel.app/api/bot
```

## تست
یه فیلم (تا حداکثر ~۲ گیگ) به بات بفرست، منتظر لینک بمون، لینک رو تو مرورگر تلویزیون باز کن.

---

## محدودیت‌ها و نکات
- پلن رایگان Vercel: حدود ۱۰۰ گیگ ترافیک در ماه — برای استفاده شخصی کافیه.
- اولین اتصال هر بار که لینک رو باز می‌کنی ممکنه ۱-۲ ثانیه طول بکشه (لاگین یوزربات).
- توکن بات و Session String رو جایی عمومی (گیت‌هاب پابلیک و…) قرار نده.
- اگه فیلم خیلی حجیمه (چند گیگ) و اینترنتت کند باشه، ممکنه گاهی پلیر باف بزنه — طبیعیه.
